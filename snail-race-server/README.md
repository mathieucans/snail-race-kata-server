🐌 Snail race server 🐌
=====

This repository a server of snail race results.

Each time a race ends, the results is available on the results url
```shell
curl localhost:8000/results
```

# Instruction 

1. Install dependencies
    ```
    npm install
    ```
2. Build application
    ```
    npm run build
    ```
   3. Start server locally
    ```
    npm run dev
    ```
